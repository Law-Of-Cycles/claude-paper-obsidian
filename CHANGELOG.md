# Changelog

## 1.1.0

- Rebuilt interface, headings, editing, and reading typography around Source Han Serif SC Variable
- Made SF Mono the first code face everywhere, including inline code, fenced code, formatting marks, and numeric markers
- Added resilient aliases for the `SourceHanSerifSC-VF` filename and its common registered family names
- Preserved Obsidian's Appearance font overrides by setting theme font variables instead of final user-facing variables
- Reworked light and dark palettes against Claude's current public design tokens
- Simplified emphasis and heading decoration for cleaner Chinese and English rhythm
- Expanded styling for tabs, sidebars, controls, settings, graph, Canvas, focus states, mobile layouts, and print
- Added installer font detection and stronger automated validation

## 1.0.0

- Initial Claude Paper release
- Added complete light and dark palettes
- Added separate English, CJK, interface, heading, and code font stacks
- Styled the full Markdown surface and Obsidian workspace chrome
- Added Windows installation and community publishing assistants
- Added bilingual documentation and three visual previews
